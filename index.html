Underscore

This is a simple static site for testing Underscore functions. The site was built using Brackets and is based on a Pluralsight course on Underscore Fundamentals. npm was used to pull down Bootstrap, jQuery and Underscore and all output is rendered to simple content div placeholders.

There are many more Underscore functions and features. Collected here are a selected subset for demo purposes. Consult the official Underscore site for the full set and documentation
